What's going on here?

In this example, after each player loads and is ready, we show a button to seek to 30 seconds and start playing. We did NOT trap any click events from the button. We attach a traditional 'onclick' event/attribute to each actual button that gets the correct player object and subsequently seeks to 30 seconds. View the source to see how this was done.
